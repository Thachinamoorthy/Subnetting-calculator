A lightweight tool for calculating IPv4 subnets. Takes an IP address and CIDR prefix and generates the essential network information needed for routing and planning.

FEATURES:

Supports standard CIDR notation (e.g., /24, /26)

Calculates Network Address and Broadcast Address

Calculates Subnet Mask and Wildcard Mask

Displays the first and last usable host IPs

Displays total available hosts

GETTING STARTED:

Clone the repository to your local machine.

Run the main file (e.g., ./subnet-calc 192.168.1.0/24).
